How Cipher Sprint works

You’re solving Caesar ciphers. Use the hot/cold meter to find the shift, then build the answer using the letter bank.

Game mechanics

  • The big text is the ciphertext (it stays the same).
  • Use ←/→ or the buttons to change the shift.
  • The preview reveals a few letters (difficulty controls how many).
  • The hot/cold indicator gets hotter as your shift approaches the correct one.
  • Fill the slots using the letter bank. Correct letters turn green.

What is a Caesar cipher?

  • It shifts each letter by a fixed amount in the alphabet.
  • Example shift 3: A→D, B→E, C→F …
  • To decode, you shift back by the same amount.
  • Your goal is to find the correct shift for each word.
