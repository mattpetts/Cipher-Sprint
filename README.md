## 🚀 Overview

A Wordle-esque game where a user is faced with a series of Caesar Ciphers (letters shifted a certain number of characters). The goal is go figure out the word based on the number of shifts.


---

## ✨ Features

- ✅ Shift mechanic. The user can shift letters in the word up and down the alphabet in order to determin the _key_ shift
- ✅ Hot & cold. a hot & cold mechanic exists to hint to the user how clow they are to the correct key
- ✅ Difficulty level. Number of shifted characters displayed to the user depends on difficulty level
- 🚧 Desktop prototype currently. Will be refactored as a react native app
- 🚧 Date based seeding. All users playing on a certain day should see the same words/ciphers
